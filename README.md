

Redux TodoApp
Overview
This TodoApp is a simple application built using Redux for state management. It allows users to add, remove, and toggle the completion status of tasks.

Features
Add new tasks
Remove existing tasks
Toggle task completion status
View list of tasks
Technologies Used
React.js
Redux
HTML
CSS
